# SmartTuneAI 智能指令优化 - 产品需求文档

## 1. 产品概述

### 1.1 产品名称
- **中文**：智能指令优化
- **英文**：SmartTuneAI
- **Slogan**：让 AI 指令更精准·更高效

### 1.2 产品定位
基于 AI 大模型的智能指令优化工具，帮助用户快速优化文本内容，提升指令质量。

### 1.3 目标用户
- 开发者/程序员
- 内容创作者
- 需要优化文案的用户

---

## 2. 功能需求

### 2.1 用户系统

| 功能 | 描述 | 优先级 |
|-----|------|-------|
| 用户注册 | 昵称 + 邮箱 + 密码注册 | P0 |
| 用户登录 | 邮箱 + 密码登录 | P0 |
| 登录状态保持 | 记住登录状态 | P1 |
| 退出登录 | 清除登录状态 | P1 |

### 2.2 核心功能

| 功能 | 描述 | 优先级 |
|-----|------|-------|
| 输入优化 | 输入文本，选择规则，一键优化 | P0 |
| 预设规则 | 错别字纠正、简洁优化、正式化表达、口语转文案 | P0 |
| 规则商店 | 浏览和使用预置规则 | P1 |
| 自定义规则 | 用户创建个性化规则 | P1 |
| 规则管理 | 编辑、删除自定义规则 | P2 |

### 2.3 使用限制

- **免费用户**：每日 10 次优化
- **后续可扩展**：会员系统，更多次数

---

## 3. 页面结构

### 3.1 品牌首页 (/)
**面向所有用户（未登录/已登录）**

**内容**：
- Hero 区域：品牌介绍 + CTA 按钮
- 核心功能：快速优化、规则商店、自定义规则
- 使用演示：优化前后对比
- 技术特点：免费、安全、开源、快速
- Footer：品牌信息

### 3.2 优化工具页 (/tool)
**仅已登录用户**

**内容**：
- 规则选择下拉框
- 文本输入区域
- 优化按钮
- 结果展示区域
- 操作：复制结果、清空

### 3.3 规则商店 (/store)
**仅已登录用户**

**内容**：
- 规则搜索
- 规则卡片列表
- 操作：应用、编辑、删除
- 新建规则按钮

### 3.4 新建规则页 (/create)
**仅已登录用户**

**内容**：
- 规则名称输入
- 规则描述
- 优化 Prompt
- 保存按钮

### 3.5 编辑规则页 (/edit/:id)
**仅已登录用户**

**内容**：
- 同新建规则，预填充已有数据

### 3.6 登录页 (/login)
**面向未登录用户**

**内容**：
- Logo + 品牌名称
- 邮箱输入
- 密码输入
- 登录按钮
- 注册链接

### 3.7 注册页 (/register)
**面向未登录用户**

**内容**：
- Logo + 品牌名称
- 昵称输入
- 邮箱输入
- 密码输入
- 确认密码
- 注册按钮
- 登录链接

## 4. 交互设计

### 4.1 页面加载
- **加载中**：显示骨架屏
- **懒加载**：页面按需加载

### 4.2 未登录拦截
- 点击"免费使用"等按钮 → 跳转登录页
- 登录成功后自动跳转回原页面

### 4.3 响应式设计
- **PC端**：顶部导航 + 左右分栏布局
- **移动端**：底部导航 + 上下堆叠布局
