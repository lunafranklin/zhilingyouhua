# SmartTuneAI 更新日志

## v0.1.0 (2025-02-04)

### ✨ 新增功能

#### 用户系统
- 用户注册功能（昵称/邮箱/密码）
- 用户登录功能（邮箱/密码）
- 登录状态保持（localStorage）
- 路由守卫（未登录拦截）

#### 品牌设计
- BrandLogo 品牌 Logo 组件
  - 包含图标、中英文品牌名
  - 支持多种尺寸
  - 蓝紫色渐变配色

- Icons 图标库
  - 16 个 SVG 图标
  - 导航、功能、特性、操作四类
  - 统一线条风格
  - 支持自定义颜色和尺寸

#### 页面功能
- 品牌首页 Landing
  - Hero 区域（品牌介绍 + CTA）
  - 核心功能区（三大功能卡片）
  - 优化演示（前后对比）
  - 技术特点展示
  - Footer 品牌信息

- 优化工具页 Home
  - 规则选择
  - 文本输入
  - 一键优化
  - 结果复制

- 规则商店页 RuleStore
  - 规则搜索
  - 规则卡片
  - 应用/编辑/删除

- 新建/编辑规则页

#### UI/UX 优化
- 骨架屏组件
  - Hero 区域
  - 功能介绍
  - 演示区域
  - 登录注册页面
  - 优化工具页面

- 懒加载优化
  - React.lazy + Suspense
  - 按需加载页面

- Toast 提示组件
- 引导弹窗组件

### 🔧 功能优化

- 响应式布局
  - PC端：顶部导航 + 左右分栏
  - 移动端：底部导航 + 上下堆叠

- 交互优化
  - 点击跳转登录
  - 登录后自动跳转
  - 骨架屏加载状态

### 🐛 问题修复

- 底部 Footer 重复问题
- 登录注册页背景渐变
- RuleStore.jsx 语法错误
- 登录后导航布局异常
