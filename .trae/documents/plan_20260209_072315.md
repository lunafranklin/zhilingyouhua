# GitHub + Vercel 上传部署计划

## 第一步：GitHub 上传

### 1.1 初始化 Git（终端执行）
```bash
cd d:\trae_old\test\zhilingyouhua_cn
git init
git add .
git commit -m "Initial commit: SmartTuneAI 智能指令优化 v0.1"
git branch -M main
```

### 1.2 创建 GitHub 仓库
- 访问 https://github.com/new
- Repository name: `Prompt-Optimizer`
- Description: `AI 智能指令优化工具 - 基于大模型的指令优化平台`
- 选择 Public
- 不要添加 README/.gitignore
- 点击 Create repository

### 1.3 推送代码
```bash
git remote add origin https://github.com/lunafranklin/Prompt-Optimizer.git
git push -u origin main
```

## 第二步：Vercel 部署

### 2.1 访问 Vercel
- 打开 https://vercel.com
- 用 GitHub 账号登录

### 2.2 导入项目
- 点击 "Add New..." → "Project"
- 选择仓库 `lunafranklin/Prompt-Optimizer`

### 2.3 配置构建
- Framework Preset: `Vite`
- Build Command: `npm run build`
- Output Directory: `dist`

### 2.4 部署
- 点击 "Deploy"
- 等待构建完成

## 项目信息
- **名称**: SmartTuneAI / Prompt-Optimizer
- **技术栈**: React + Vite + TailwindCSS
- **功能**: AI 指令优化、规则商店、用户系统
