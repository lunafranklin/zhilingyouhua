## 指令优化工具 - 开发计划

### 📋 项目概述

开发一个在线 AI 提示词优化工具，帮助用户优化输入的指令内容。用户可以选择预置规则或自定义规则，系统调用大模型进行智能优化。

---

### 🛠️ 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端框架 | React + Vite | 快速开发，热更新 |
| UI 样式 | Tailwind CSS | 轻量、灵活、美观 |
| 状态管理 | React Hooks | 简单够用 |
| 后端 | Node.js + Express | 处理 API 请求 |
| 大模型 | Kimi API / 智谱 AI | 免费额度充足 |
| 本地存储 | localStorage | 存储用户规则 |

---

### 🎯 功能模块

#### 1. 首页 / 优化工具
- 输入框（支持粘贴、输入）
- 规则选择下拉菜单
- "开始优化"按钮
- 结果展示区域
- 复制结果功能

#### 2. 规则商店
- 规则卡片列表展示
- 搜索规则功能
- 点击卡片进入规则详情/编辑
- "选用此规则"按钮

#### 3. 规则管理
- 新建规则（支持模板参考）
- 编辑规则
- 删除规则
- 规则数据结构：名称 + 简介 + 描述（提示词）

#### 4. 新手引导
- 首次使用弹窗引导
- 三步上手教程

---

### 📝 开发步骤

#### 第一阶段：项目初始化
1. 初始化 React + Vite 项目
2. 配置 Tailwind CSS
3. 创建项目目录结构
4. 搭建基础页面框架

#### 第二阶段：前端界面开发
1. 开发首页（优化工具界面）
2. 开发规则商店页面
3. 开发规则详情/编辑页面
4. 开发新建规则页面
5. 优化样式，提升用户体验

#### 第三阶段：后端 API 开发
1. 搭建 Express 服务
2. 集成大模型 API（Kimi/智谱）
3. 实现优化接口

#### 第四阶段：功能完善
1. 实现规则本地存储（localStorage）
2. 添加新手引导弹窗
3. 优化交互细节
4. 测试和修复 Bug
