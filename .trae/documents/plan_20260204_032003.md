## 登录注册系统

### 1. 创建认证服务
- `frontend/src/services/auth.js`
- 使用 localStorage 模拟存储
- 支持登录、注册、退出、获取当前用户
- 预留 Supabase/阿里云接口

### 2. 创建 AuthContext
- `frontend/src/contexts/AuthContext.jsx`
- 全局管理用户登录状态
- 提供 login, register, logout, user 等方法

### 3. 创建登录页面
- `frontend/src/pages/Login.jsx`
- 邮箱 + 密码登录
- 底部有"还没有账号？去注册"链接

### 4. 创建注册页面
- `frontend/src/pages/Register.jsx`
- 昵称 + 邮箱 + 密码 + 确认密码
- 底部有"已有账号？去登录"链接

### 5. 更新路由
- App.jsx 添加 /login 和 /register 路由
- 更新顶部导航显示逻辑
- 添加路由守卫（未登录访问跳转到登录页）

### 6. 更新品牌首页
- 未登录：隐藏顶部菜单 + 底部导航
- 右上角显示"登录/注册"按钮
- 点击功能按钮时拦截 + 提示

### 7. 创建 Toast 提示组件
- `frontend/src/components/Toast.jsx`
- 顶部弹出的轻量级提示
